## SMS 三竹資訊
因為mitake-larvel 只能用於laravel 6以前，所以需要把程式碼拆解一下，

### Installation
```shell
$ composer require minchao/mitake-php
```

### Configuration
```dotenv
MITAKE_USERNAME=username
MITAKE_PASSWORD=password
```
